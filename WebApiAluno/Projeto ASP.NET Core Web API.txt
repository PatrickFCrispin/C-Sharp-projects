Projeto ASP.NET Core Web API

Para testar:

Postman:
1. Rodar o programa no VS;
2. Abrir o Postman;
3. Em API (Postman), testar a sua requisição selecionando o verbo HTTP desejado e atribuindo o body se necessário (Post, Put).

Insomnia
1. Rodar o programa no VS;
2. Abrir o Insomnia;
3. Criar uma nova Request ou Requests selecionando o verbo HTTP desejado e atribuindo o body se necessário (Post, Put).
